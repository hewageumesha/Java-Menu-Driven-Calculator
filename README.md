## 📘 Java Menu-Driven Calculator

### 🚀 Overview

This project is a console-based **Java Menu-Driven Calculator** that performs basic and advanced arithmetic operations. It is designed using core Java concepts such as methods, conditionals, loops, and exception handling.

⭐ This project was developed as my **first task at Saiket Systems as a Java Developer Intern**.

---

### 💻 Features

* ✅ Menu-driven interface
* ✅ Basic operations (Addition, Subtraction, Multiplication, Division, Power)
* ✅ Exception handling (e.g., division by zero)
* ✅ Input validation for user-friendly experience
* ✅ Reusable methods and clean structure
* ✅ Continuous execution using loop

---

### 🛠️ Technologies Used

* Java
* Scanner (for user input)
* Core Java Concepts (Methods, Conditionals, Loops, Exception Handling)

---

### ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Java-Menu-Driven-Calculator.git
```

2. Navigate to the project folder

```bash
cd Java-Menu-Driven-Calculator
```

3. Compile the program

```bash
javac AdvancedCalculator.java
```

4. Run the program

```bash
java AdvancedCalculator
```
